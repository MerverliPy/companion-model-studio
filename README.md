# Companion Model Studio

Local-first platform for creating, training, and improving a user-owned AI companion.

## V1
- Create a companion
- Pick avatar + template + skill packs
- Connect to a local model runtime
- Run lessons/evals
- Track progress and badges
- Chat locally

## Development workflow
This repo is built with OpenCode using:
orchestrator -> builder -> validator -> builder(if fail) -> validator -> shipper
